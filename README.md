# Animator Portfolio Website

一个适合动画师、动效设计师展示作品的静态个人网站模板。

## 包含页面

- `index.html` 首页
- `works.html` 作品集
- `about.html` 关于我

## 项目结构

```text
.
├── assets/
│   ├── icons/
│   ├── images/
│   └── videos/
├── about.html
├── index.html
├── script.js
├── style.css
└── works.html
```

## 如何替换成你自己的内容

1. 把页面里的 `Luna Lin` 改成你的名字
2. 把邮箱和社交链接改成你的联系方式
3. 把你的视频放到 `assets/videos/`
4. 修改作品标题、介绍文案和关于页内容

建议视频文件名：

- `your-best-work.mp4`
- `work1.mp4`
- `work2.mp4`
- `work3.mp4`

## 本地预览

直接双击 `index.html` 即可预览。

## 部署

可以直接部署到：

- Vercel
- Netlify
- GitHub Pages

## 说明

当前仓库是纯静态网站，不依赖额外安装步骤。
