# Animator Portfolio Website

“睿子动画”的静态个人网站项目，适合展示 AI 时代的动画、动效与视觉叙事作品。

## 包含页面

- `index.html` 首页
- `works.html` 作品集
- `about.html` 关于我

## 项目结构

```text
.
├── assets/
│   ├── icons/
│   ├── images/
│   └── videos/
├── about.html
├── index.html
├── script.js
├── style.css
└── works.html
```

## 如何替换成你自己的内容

1. 把占位邮箱和社交链接改成你的真实联系方式
2. 把你的视频放到 `assets/videos/`
3. 修改作品标题、介绍文案和关于页内容
4. 按需替换站内占位海报图

建议视频文件名：

- `your-best-work.mp4`
- `work1.mp4`
- `work2.mp4`
- `work3.mp4`

## 本地预览

直接双击 `index.html` 即可预览。

## 部署

可以直接部署到：

- Vercel
